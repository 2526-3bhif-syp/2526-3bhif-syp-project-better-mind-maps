== Ergebnis

Auf Basis dieser Antworten wurde folgende Implementierung umgesetzt:

* Es wird eine kleine Datenbank verwendet, damit alles gespeichert bleibt. Dafür gibt es eine eigene Klasse `DatabaseManager`.
* Knoten (Nodes) können jetzt Text und eine Position (x und y) speichern. Außerdem können Knoten gelöscht werden.
* Es gibt Funktionen, um Daten zu speichern, zu aktualisieren, zu löschen und zu laden.
* Mit Rechtsklick kann man einen neuen Unterknoten erstellen, Text bearbeiten und Knoten löschen. Verbindungen zwischen Knoten werden angezeigt und die Navigation per Tastatur ist möglich.
* Man kann neue Knoten hinzufügen, Texte ändern und Knoten löschen.
