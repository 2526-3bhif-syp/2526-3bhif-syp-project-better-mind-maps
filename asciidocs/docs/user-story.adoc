= User Stories

* Mindmap creation (Issue #1)
** As a student, I want to create a new mind map so that I can collect and organize ideas.

* Mindmap editing (Issue #2)
** As a student, I want to edit my mind map so that I can add, edit and delete nodes.

* Account management (Issue #3)
** As a student, I want to create an account so that I can access my mind maps on different devices.

* Mindmap management (Issue #4)
** As a student, I want to manage my mind maps so that I can save, load and delete them.

* Presentation (Issue #5)
** As a professor, I want a presentation mode so that I can present mind maps clearly in fullscreen.

* Zoom / Navigation (Issue #6)
** As a student, I want to zoom and pan the mind map so that I can work with large maps easily.

* Move / Arrange Nodes (Issue #7)
** As a student, I want to move nodes freely so that I can organize the map visually.

* Connect Nodes (Issue #8)
** As a student, I want to create connections between nodes so that I can show relationships.

* Style Nodes (Issue #9)
** As a student, I want to style nodes (color/size) so that I can highlight important information.

* Open Mind Map (Issue #10)
** As a student, I want to open existing mind maps so that I can continue working on them.
