== Ergebnis

Auf Basis dieser Antworten wurde folgende Implementierung umgesetzt:

* *Persistenz:* SQLite via `org.xerial:sqlite-jdbc`, neuer `DatabaseManager` für
  Verbindungsverwaltung und Schema-Initialisierung
* *Datenmodell:* `setText`, `setXCoordinate`, `setYCoordinate` in `Node` ergänzt;
  `removeNode` in `MindMap` ergänzt
* *Repository:* vollständige SQLite-Implementierung (`save`, `saveNode`, `updateNode`,
  `deleteNode`, `loadAll`)
* *Service:* `addNode`, `updateNodeText`, `deleteNode` (mit Cascade) hinzugefügt
* *UI:* Rechtsklick-Kontextmenü (Add Child Node / Edit Text / Delete Node),
  Kanten-Rendering auf demselben `Pane`, Tastaturnavigation
* *Tests:* 17 Unit-Tests in `MindMapServiceTest` mit temporärer SQLite-Datenbank
